# FitQA

`Flutter`기반의 FitQA 모바일 어플리케이션입니다. 
FitQA는 비대면 헬스 자세 피드백 플랫폼으로 `Android`, `iOS` 에서 동작합니다.

## 주요기능
- google / kakao / naver OAuth2 로그인
- 사용자의 운동 동영상을 통한 전문 트레이너에게 헬스 자세 피드백 요청
- 다른사람의 피드백을 보고 배울 수 있는 피드
- 사생활 보호를 위한 비공개 기능

## How to run
### 디버그
```
flutter run
```

### 릴리즈
```
flutter run --release
```
